# ‚ùå‚≠ï Tic-Tac-Toe Game

A classic two-player Tic-Tac-Toe (Noughts and Crosses) game implemented as a command-line interface application in Python. It features an intuitive numerical input system and clear ASCII art visuals.

## ‚ú® Features

* *Two-Player Mode:* Play against a friend locally on the same terminal.
* *Intuitive Input:* Players enter a number (1-9) to place their mark (X or O).
* *Clear Board Mapping:* The board is numbered 1 through 9, corresponding directly to the input keys for easy play.
* *Win Detection:* Automatically checks for winning conditions and draws.
* *Aesthetic Output:* Uses *pyfiglet* for a stylish command-line heading.
